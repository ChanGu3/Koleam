# Koleam

> ---
>
> ### Self Video Streaming Service
>
> #### Users
>
> - Admin
>   - upload, update, delete Streams, view analytics, and manage other members.
> - Users
>   - watch Streams
>
> #### Streams
>
> - Titles: the Intellectual Property (IP) of the product
> - Installment: A list of episodes or movies
> - Stream: the episode or movie themselves
>

## Setup

using nodejs package.json workspaces makes installing dependencies easier for both server and web client.
> ---
>
> ### Pre-Requisite
>
> - NodeJS
>
>
> ### Install Dependencies
>
> - in the CLI enter `npm install` or `npm i` in the [full-stack](full-stack) directory
>
